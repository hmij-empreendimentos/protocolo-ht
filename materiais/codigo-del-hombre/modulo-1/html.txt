      <div class="acc-body-inner">
        <p>Existe una idea popular de que dormir es "apagar el cerebro". La investigación de las últimas décadas demuestra lo contrario: durante el sueño, el cerebro ejecuta funciones biológicas que <strong>no puede realizar mientras estás despierto</strong>.</p>
        <p>Matthew Walker, profesor de neurociencia en la Universidad de California, Berkeley, y director del Center for Human Sleep Science, lleva más de 25 años estudiando el sueño. Su trabajo, sintetizado en el libro <em>Why We Sleep</em> (2017), documenta que el sueño no es un descanso pasivo — es un período de actividad biológica intensa, organizada y necesaria.</p>

        <p>Las principales funciones que el cerebro y el cuerpo ejecutan durante el sueño:</p>

        <div class="funcion-card">
          <div class="funcion-emoji">💾</div>
          <div class="funcion-info">
            <div class="funcion-titulo">Consolidación de memoria</div>
            <div class="funcion-desc"><strong>El cerebro guarda lo aprendido.</strong> Durante el sueño, la información del día se transfiere de la memoria de corto plazo al almacenamiento de largo plazo. Sin sueño suficiente, el aprendizaje del día anterior se pierde.</div>
          </div>
        </div>

        <div class="funcion-card">
          <div class="funcion-emoji">🧹</div>
          <div class="funcion-info">
            <div class="funcion-titulo">Limpieza cerebral</div>
            <div class="funcion-desc"><strong>El sistema glinfático actúa.</strong> Descubierto en 2013 por la investigadora Maiken Nedergaard, de la University of Rochester, este sistema elimina desechos metabólicos acumulados durante el día. Solo opera durante el sueño profundo.</div>
          </div>
        </div>

        <div class="funcion-card">
          <div class="funcion-emoji">🔧</div>
          <div class="funcion-info">
            <div class="funcion-titulo">Reparación de tejidos</div>
            <div class="funcion-desc"><strong>El cuerpo se regenera.</strong> Durante el sueño profundo se libera hormona del crecimiento, que participa en la reparación de tejidos musculares, óseos y celulares.</div>
          </div>
        </div>

        <div class="funcion-card">
          <div class="funcion-emoji">❤️</div>
          <div class="funcion-info">
            <div class="funcion-titulo">Regulación emocional</div>
            <div class="funcion-desc"><strong>El cerebro procesa el día.</strong> Las experiencias emocionales se reorganizan durante el sueño REM. Un sueño insuficiente reduce la capacidad de regular emociones al día siguiente.</div>
          </div>
        </div>

        <div class="funcion-card">
          <div class="funcion-emoji">🛡️</div>
          <div class="funcion-info">
            <div class="funcion-titulo">Soporte al sistema inmune</div>
            <div class="funcion-desc"><strong>Las defensas se fortalecen.</strong> Durante el sueño se producen proteínas llamadas citoquinas, que participan en la respuesta inmune. Investigación documenta menor resistencia a infecciones tras períodos de sueño insuficiente.</div>
          </div>
        </div>

        <div class="quote">
          <div class="quote-text">"El sueño es el cimiento sobre el cual se construyen todos los aspectos de la salud física y mental. Cuando se compromete el sueño, se compromete todo lo demás."</div>
          <cite class="quote-cite">— Dr. Matthew Walker, UC Berkeley</cite>
        </div>

        <div class="info-box">
          <p><strong>El insight central:</strong> dormir mal no es solo "estar cansado al día siguiente". Es perder funciones biológicas que el cerebro y el cuerpo realizan exclusivamente durante el sueño — funciones que afectan claridad mental, estado de ánimo y energía sostenida.</p>

        <p>El sueño no es un estado uniforme. Cada noche, tu cuerpo atraviesa entre <strong>4 y 6 ciclos completos</strong>, cada uno de aproximadamente 90 minutos. Esta estructura fue descrita inicialmente por Nathaniel Kleitman y Eugene Aserinsky en la Universidad de Chicago, en investigaciones de las décadas de 1950 y 1960 que fundaron la ciencia moderna del sueño.</p>
        <p>Cada ciclo está compuesto por 4 fases diferentes, cada una con funciones biológicas específicas:</p>

        <div class="fases-grid">
          <div class="fase-card">
            <span class="fase-num">N1</span>
            <div class="fase-nombre">Transición</div>
            <div class="fase-tiempo">1–5 min</div>
          </div>
          <div class="fase-card">
            <span class="fase-num">N2</span>
            <div class="fase-nombre">Sueño Ligero</div>
            <div class="fase-tiempo">10–25 min</div>
          </div>
          <div class="fase-card">
            <span class="fase-num">N3</span>
            <div class="fase-nombre">Sueño Profundo</div>
            <div class="fase-tiempo">20–40 min</div>
          </div>
          <div class="fase-card">
            <span class="fase-num">REM</span>
            <div class="fase-nombre">Movimiento Ocular</div>
            <div class="fase-tiempo">10–60 min</div>
          </div>
        </div>

        <p><strong>Sueño Profundo (N3).</strong> Es la fase donde ocurre la mayor parte de la recuperación física. El cuerpo libera hormona del crecimiento, repara tejidos y refuerza el sistema inmune. Si esta fase es interrumpida o reducida, la sensación de cansancio al despertar puede aparecer incluso después de 8 horas de sueño.</p>

        <p><strong>Sueño REM (Rapid Eye Movement).</strong> Es la fase donde ocurre la mayor parte de la recuperación mental. El cerebro procesa emociones del día, consolida memorias y reorganiza información. La mayoría de los sueños vívidos ocurren aquí. Una reducción del REM se asocia con desregulación emocional y problemas de memoria al día siguiente.</p>

Esto explica el cansancio al día siguiente incluso tras varias horas de "sueño" después de beber. Este dato es educativo — no constituye recomendación clínica.</p>

      <div class="acc-body-inner">
        <p>Una persona puede dormir 8 horas y despertar cansada. Otra puede dormir 6 horas y despertar relativamente recuperada. La diferencia, según la investigación contemporánea sobre sueño, está en lo que se llama <strong>arquitectura del sueño</strong> — la distribución y completitud de las fases durante la noche.</p>
        <p>La cantidad de horas es importante, pero por sí sola no garantiza descanso. Un sueño fragmentado de 8 horas, con interrupciones que cortan los ciclos antes de completarse, produce menos recuperación que un sueño consolidado de 6.5 horas con ciclos completos.</p>

        <div class="quote">
          <div class="quote-text">"No todas las horas de sueño son iguales. La calidad — medida por la consolidación de ciclos completos — determina qué tanto te recuperas durante la noche."</div>
          <cite class="quote-cite">— Investigación sobre arquitectura del sueño</cite>
        </div>

        <div class="senales-grid">
          <div class="senales-card senales-mala">
            <div class="senales-label">– Señales de baja calidad</div>
            <ul class="senales-list">

              <li>Dependes de alarmas fuertes para despertar</li>

              <li>Te sientes alerta dentro de los primeros 30 minutos</li>

        <span class="acc-section-label">El reloj interno</span>
        <div class="acc-titulo">El ritmo circadiano</div>
      </div>
      <div class="acc-right">
        <div class="acc-arrow-btn"><svg viewBox="0 0 12 12"><polyline points="2,4 6,8 10,4"/></svg></div>

    <div class="acc-body" id="body-s4">
      <div class="acc-body-inner">
        <p>Dentro del cerebro existe una estructura llamada <strong>núcleo supraquiasmático</strong>, ubicada en el hipotálamo. Funciona como un "reloj maestro" que regula el ciclo de 24 horas — sueño y vigilia, alerta y descanso. Este sistema se conoce como <strong>ritmo circadiano</strong> (del latín <em>circa diem</em>, "alrededor de un día").</p>
        <p>El ritmo circadiano fue caracterizado a profundidad por investigadores como Charles Czeisler (Harvard Medical School) y Mary Carskadon (Brown University), pioneros en cronobiología — el estudio científico de los ritmos biológicos. La luz solar es el principal sincronizador de este reloj interno.</p>

        <p>Una jornada típica de 24 horas, según la cronobiología:</p>

        <div class="reloj-card">
          <div class="reloj-hora">6–8h</div>
          <div class="reloj-info"><strong>Activación matinal.</strong> El cuerpo inicia naturalmente el despertar, aumenta la temperatura corporal y se prepara para la actividad del día.</div>
        </div>

        <div class="reloj-card">
          <div class="reloj-hora">10–14h</div>
          <div class="reloj-info"><strong>Pico de alerta.</strong> El período de mayor rendimiento cognitivo del día para la mayoría de las personas. Trabajo profundo y tareas complejas funcionan mejor aquí.</div>
        </div>

        <div class="reloj-card">
          <div class="reloj-hora">14–16h</div>
          <div class="reloj-info"><strong>Bajón natural.</strong> Es normal una reducción de la alerta a media tarde. No es señal de problema — es parte del ciclo natural.</div>
        </div>

        <div class="reloj-card">
          <div class="reloj-hora">18–20h</div>
          <div class="reloj-info"><strong>Inicio del modo descanso.</strong> El cuerpo comienza a desacelerar gradualmente. La temperatura corporal empieza a bajar.</div>
        </div>

        <div class="reloj-card">
          <div class="reloj-hora">21–23h</div>
          <div class="reloj-info"><strong>Preparación para el sueño.</strong> Si la exposición lumínica es adecuada, el cuerpo prepara naturalmente la transición hacia el sueño.</div>
        </div>

        <div class="reloj-card">
          <div class="reloj-hora">2–4h</div>
          <div class="reloj-info"><strong>Sueño más profundo.</strong> En esta franja se concentra la mayor parte del sueño profundo de la noche — la fase de máxima recuperación física.</div>
        </div>

        <p>Los <strong>3 principales disruptores del ritmo circadiano</strong> identificados por la investigación:</p>

        <div class="disruptor-card">
          <div class="disruptor-top">
            <div class="disruptor-emoji">📱</div>
            <div class="disruptor-titulo">Luz azul de pantallas en la noche</div>
          </div>
          <div class="disruptor-desc"><strong>Confunde al cerebro.</strong> La luz azul emitida por celulares, computadoras y televisores en horarios nocturnos envía al cerebro la señal de que aún es de día — alterando la preparación natural para el sueño.</div>
        </div>

        <div class="disruptor-card">
          <div class="disruptor-top">
            <div class="disruptor-emoji">⏰</div>
            <div class="disruptor-titulo">Horarios irregulares</div>
          </div>
          <div class="disruptor-desc"><strong>El reloj no se calibra.</strong> Acostarse y despertarse a horas muy variables impide que el sistema circadiano establezca un ritmo estable. Walker compara esto con "viajar a una zona horaria distinta cada noche".</div>
        </div>

        <div class="disruptor-card">
          <div class="disruptor-top">
            <div class="disruptor-emoji">☀️</div>
            <div class="disruptor-titulo">Falta de luz solar matinal</div>
          </div>
          <div class="disruptor-desc"><strong>Sin señal de inicio del día.</strong> Andrew Huberman, profesor de la Stanford School of Medicine, documenta que la exposición a luz solar dentro de los primeros 30–60 minutos tras despertar es uno de los anclajes más fuertes del ritmo circadiano.</div>
        </div>

        <div class="info-box">
          <p><strong>Importante:</strong> los tiempos para que el ritmo circadiano se reajuste varían entre personas según múltiples factores individuales — edad, hábitos previos, contexto laboral, exposición lumínica. La constancia en pequeños ajustes diarios suele tener más impacto que cambios drásticos puntuales.</p>
        </div>
      </div>

<script>
  var openItems = {};
  var completed = {};
  var totalSections = 6;

  function toggle(id) {
    var item = document.getElementById(id);
    var body = document.getElementById('body-' + id);
    var isOpen = openItems[id];
    if (isOpen) {
      body.style.maxHeight = '0px';
      item.classList.remove('open');
      openItems[id] = false;
    } else {
      body.style.maxHeight = body.scrollHeight + 'px';
      item.classList.add('open');
      openItems[id] = true;
      if (!completed[id]) {
        setTimeout(function() { markRead(id); }, 1800);
      }
    }
    updateXP();
  }

  function markRead(id) {
    if (completed[id]) return;
    completed[id] = true;
    document.getElementById(id).classList.add('completed');
    updateXP();
  }

  function updateXP() {
    var count = Object.keys(completed).length;
    var pct = Math.round((count / totalSections) * 100);
    document.getElementById('xpFill').style.width = pct + '%';
    document.getElementById('xpPct').textContent = pct + '%';
  }

  function completarCap() {
    for (var i = 1; i <= totalSections; i++) {
      completed['s' + i] = true;
      document.getElementById('s' + i).classList.add('completed');
    }
    document.getElementById('xpFill').style.width = '100%';
    document.getElementById('xpPct').textContent = '100%';
    var toast = document.getElementById('toast');
    toast.classList.add('show');
    setTimeout(function() { toast.classList.remove('show'); }, 3000);
  }

  function toggleCheck(el) {
    el.classList.toggle('checked');
  }

  setTimeout(function() {
    document.getElementById('xpFill').style.width = '0%';
  }, 100);
</script>
