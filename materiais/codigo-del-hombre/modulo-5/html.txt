      </div>
      <div class="acc-right">
        <div class="acc-arrow-btn"><svg viewBox="0 0 12 12"><polyline points="2,4 6,8 10,4"/></svg></div>

    <div class="acc-body" id="body-s1">
      <div class="acc-body-inner">
        <p>Uno de los factores ambientales más estudiados — y más subestimados — es la temperatura de la habitación durante la noche. El cuerpo humano experimenta una <strong>bajada natural de temperatura corporal</strong> como parte del proceso de entrada al sueño. Cuando el ambiente es muy cálido, esa bajada se ve dificultada, y el resultado puede ser sueño fragmentado y menos consolidado.</p>
        <p>Matthew Walker, en <em>Why We Sleep</em> (2017), describe la temperatura del cuarto como una de las variables ambientales con efecto más consistente sobre la calidad del descanso. Los estudios de cronobiología sugieren un rango general de temperatura que tiende a apoyar mejor el sueño.</p>

        <div class="stats-row">
          <div class="stat-card">
            <span class="stat-num">18–20°C</span>
            <div class="stat-label">Rango general sugerido (65–68°F)</div>
          </div>
          <div class="stat-card">
            <span class="stat-num">22°C+</span>
            <div class="stat-label">Suele resultar incómodo para muchos</div>
          </div>
        </div>

        <div class="quote">
          <div class="quote-text">"Una habitación demasiado caliente es uno de los disruptores más comunes y silenciosos del sueño. La temperatura del ambiente afecta directamente la calidad del descanso."</div>
          <cite class="quote-cite">— Dr. Matthew Walker, UC Berkeley</cite>
        </div>

        <p>Estrategias educativas para gestionar la temperatura del dormitorio:</p>

        <div class="solucion-card">
          <div class="solucion-emoji">❄️</div>
          <div class="solucion-info">
            <div class="solucion-titulo">Aire acondicionado o ventilador</div>
            <div class="solucion-desc">Si tienes acceso, configurar la temperatura dentro del rango sugerido antes de acostarte permite que el ambiente esté preparado cuando entres a la habitación.</div>
          </div>
        </div>

        <div class="solucion-card">
          <div class="solucion-emoji">🪟</div>
          <div class="solucion-info">
            <div class="solucion-titulo">Ventana abierta cuando el clima lo permite</div>
            <div class="solucion-desc">En climas templados o frescos, dejar entrar aire fresco durante la noche puede ser suficiente para mantener una temperatura adecuada.</div>
          </div>
        </div>

        <div class="solucion-card">
          <div class="solucion-emoji">🛏️</div>
          <div class="solucion-info">
            <div class="solucion-titulo">Ropa de cama transpirable</div>
            <div class="solucion-desc">Materiales como algodón o lino permiten mejor disipación del calor que tejidos sintéticos. La elección de la ropa de cama puede compensar parcialmente un ambiente más cálido.</div>
          </div>
        </div>

        <div class="solucion-card">
          <div class="solucion-emoji">🚿</div>
          <div class="solucion-info">
            <div class="solucion-titulo">Ducha tibia 1–2 horas antes</div>
            <div class="solucion-desc">Un metaanálisis publicado en <em>Sleep Medicine Reviews</em> (Haghayegh et al., 2019, UT Austin) documenta que una ducha tibia 1 a 2 horas antes de acostarse se asocia con menor tiempo para conciliar el sueño — paradójicamente, ayuda a que la temperatura corporal baje después.</div>
          </div>
        </div>

        <div class="info-box">
          <p><strong>El principio detrás:</strong> el cuerpo "lee" la temperatura del entorno como una señal. Un ambiente fresco apoya la transición natural al sueño. Un ambiente cálido la dificulta — incluso si la persona no se da cuenta conscientemente.</p>
        </div>

  <div class="acc-item" id="s2">
    <button class="acc-trigger" onclick="toggle('s2')">
      <div class="acc-icon-wrap">🌑</div>
      <div class="acc-titles">
        <span class="acc-section-label">Oscuridad</span>
        <div class="acc-titulo">Por qué la oscuridad total importa</div>
      </div>
      <div class="acc-right">
        <div class="acc-arrow-btn"><svg viewBox="0 0 12 12"><polyline points="2,4 6,8 10,4"/></svg></div>
        <span class="acc-check">✅</span>
      </div>
    </button>
    <div class="acc-body" id="body-s2">
      <div class="acc-body-inner">
        <p>El cuerpo humano evolucionó durante miles de generaciones bajo un patrón de luz y oscuridad relativamente estable: día claro, noche oscura. La iluminación artificial cambió ese patrón hace apenas un siglo — un suspiro en la escala evolutiva. Por eso el sistema circadiano sigue siendo muy sensible a cualquier exposición lumínica nocturna.</p>
        <p>Una investigación coordinada por Joshua Gooley en el Brigham and Women's Hospital (Harvard Medical School, 2011) documentó que <strong>incluso luz tenue durante la noche puede interferir en la señal natural de descanso</strong> que el cuerpo prepara en esas horas. Estamos hablando de niveles bajos — el LED de un cargador, una luz piloto, el resplandor de una pantalla.</p>

        <p>El objetivo es lo que algunos investigadores llaman <em>"oscuridad de cueva"</em>: un nivel de oscuridad tal que, con los ojos abiertos, no puedas distinguir tu mano frente a tu cara.</p>

        <p>Estrategias educativas para alcanzar oscuridad efectiva:</p>

        <div class="solucion-card">
          <div class="solucion-emoji">🪟</div>
          <div class="solucion-info">
            <div class="solucion-titulo">Cortinas o persianas blackout</div>
            <div class="solucion-desc"><strong>La inversión con mayor impacto.</strong> Bloquean prácticamente toda la luz exterior — incluyendo luces de la calle, faros, anuncios nocturnos.</div>
          </div>
        </div>

        <div class="solucion-card">
          <div class="solucion-emoji">💡</div>
          <div class="solucion-info">
            <div class="solucion-titulo">Cubrir LEDs y luces piloto</div>
            <div class="solucion-desc">Pequeñas luces de cargadores, televisores, módems, despertadores digitales. Una cinta opaca o moverlas fuera del campo visual nocturno es suficiente.</div>
          </div>
        </div>

        <div class="solucion-card">
          <div class="solucion-emoji">😴</div>
          <div class="solucion-info">
            <div class="solucion-titulo">Antifaz para dormir</div>
            <div class="solucion-desc">Solución económica e inmediata cuando no es posible bloquear toda la luz del ambiente. Especialmente útil en habitaciones que no se pueden oscurecer completamente.</div>
          </div>
        </div>

        <div class="solucion-card">
          <div class="solucion-emoji">📱</div>
          <div class="solucion-info">
            <div class="solucion-titulo">Celular boca abajo o fuera del cuarto</div>
            <div class="solucion-desc">La pantalla que se ilumina con cada notificación es una fuente de luz nocturna que muchas personas ignoran. Dejarlo fuera del dormitorio elimina la fuente — y otros disruptores asociados.</div>
          </div>
        </div>

        <div class="warn-box">
          <div class="warn-box-icon">⚠️</div>
          <p><strong>Si te levantas al baño en la noche:</strong> evita encender luces blancas brillantes. Una luz tenue, idealmente con tonalidad cálida (amarillenta o anaranjada), interfiere menos con la transición de regreso al sueño que una luz blanca intensa.</p>
        </div>
      </div>

  <div class="acc-item" id="s3">
    <button class="acc-trigger" onclick="toggle('s3')">

      <div class="acc-titles">

        <div class="acc-titulo">El silencio y sus alternativas</div>
      </div>
      <div class="acc-right">
        <div class="acc-arrow-btn"><svg viewBox="0 0 12 12"><polyline points="2,4 6,8 10,4"/></svg></div>

      </div>
    </button>
    <div class="acc-body" id="body-s3">
      <div class="acc-body-inner">
        <p>El cerebro nunca "apaga" completamente la audición. Incluso durante el sueño profundo, sigue procesando sonidos del entorno — una adaptación evolutiva que permitía a nuestros antepasados despertar ante amenazas. En el contexto moderno, esto significa que los ruidos del ambiente pueden fragmentar el sueño <em>aunque la persona no recuerde haberse despertado</em>.</p>
        <p>La investigación documenta que ruidos nocturnos repetidos generan lo que se llama <strong>micro-despertares</strong>: transiciones muy breves desde fases más profundas del sueño hacia fases más superficiales, que se asocian con menor recuperación al despertar.</p>

        <p>Tres estrategias educativas frecuentemente útiles:</p>

        <div class="ruido-grid">
          <div class="ruido-card">
            <span class="ruido-emoji">🔇</span>
            <div class="ruido-nombre">Tapones de oídos</div>
            <div class="ruido-desc">Económico, accesible y efectivo</div>
          </div>
          <div class="ruido-card">
            <span class="ruido-emoji">🌊</span>
            <div class="ruido-nombre">Ruido blanco</div>
            <div class="ruido-desc">Sonido constante que enmascara</div>
          </div>
          <div class="ruido-card">
            <span class="ruido-emoji">🎧</span>
            <div class="ruido-nombre">Auriculares de sueño</div>
            <div class="ruido-desc">Más cómodos para algunos</div>
          </div>
        </div>

        <p>Sobre el concepto de "ruido blanco" y similares:</p>

        <div class="solucion-card">
          <div class="solucion-emoji">🌊</div>
          <div class="solucion-info">
            <div class="solucion-titulo">Ruido blanco</div>
            <div class="solucion-desc"><strong>Sonido constante de amplio espectro</strong> que enmascara ruidos variables del entorno (autos, ladridos, voces). El cerebro deja de "engancharse" a sonidos puntuales y mantiene el descanso más estable.</div>
          </div>
        </div>

        <div class="solucion-card">
          <div class="solucion-emoji">🌧️</div>
          <div class="solucion-info">
            <div class="solucion-titulo">Ruido rosa o sonidos de naturaleza</div>
            <div class="solucion-desc">Versiones más suaves que el ruido blanco — lluvia, cascada, viento. Algunas personas las prefieren porque resultan más placenteras subjetivamente.</div>
          </div>
        </div>

        <div class="solucion-card">
          <div class="solucion-emoji">🌀</div>
          <div class="solucion-info">
            <div class="solucion-titulo">Ventilador</div>
            <div class="solucion-desc">Doble función práctica: enfría el ambiente y produce un ruido constante de fondo. Para quienes lo toleran bien, suele ser la solución más simple.</div>
          </div>
        </div>

        <div class="warn-box">
          <div class="warn-box-icon">⚠️</div>
          <p><strong>Lo que no funciona como "ruido para dormir":</strong> dejar la televisión encendida o reproducir música con letra. El cerebro intenta procesar el contenido — historias, conversaciones, melodías con palabras — y eso mantiene actividad cognitiva que dificulta el descanso profundo. Si vas a usar audio, prefiere sonidos sin información (ruido blanco, lluvia, sonidos ambientales puros).</p>
        </div>
      </div>
    </div>

  <!-- S4 - ASOCIACION CAMA -->
  <div class="acc-item" id="s4">
    <button class="acc-trigger" onclick="toggle('s4')">
      <div class="acc-icon-wrap">🛏️</div>
      <div class="acc-titles">
        <span class="acc-section-label">Asociación mental</span>
        <div class="acc-titulo">La cama como señal de sueño</div>
      </div>
      <div class="acc-right">
        <div class="acc-arrow-btn"><svg viewBox="0 0 12 12"><polyline points="2,4 6,8 10,4"/></svg></div>
        <span class="acc-check">✅</span>
      </div>
    </button>
    <div class="acc-body" id="body-s4">
      <div class="acc-body-inner">
        <p>El cerebro construye <strong>asociaciones automáticas</strong> entre lugares y actividades. Si trabajas, ves televisión, comes y respondes mensajes en la cama, el cerebro asocia esa cama con "estar despierto y activo" — y eso dificulta la transición al sueño cuando finalmente intentas dormir.</p>
        <p>Esta lógica viene de la <em>terapia cognitivo-conductual para el insomnio</em>, desarrollada inicialmente por Richard Bootzin en la Northwestern University en la década de 1970, y ampliamente validada desde entonces. Una de sus reglas centrales: <strong>la cama debe asociarse principalmente con dormir</strong>.</p>

        <div class="quote">
          <div class="quote-text">"Si pasas tiempo despierto y activo en tu cama, le enseñas a tu cerebro que la cama es un lugar para estar despierto. Si solo estás ahí cuando duermes, le enseñas que la cama es para dormir."</div>
          <cite class="quote-cite">— Principio de control de estímulos</cite>
        </div>

        <p>Lo que la literatura sugiere evitar en la cama:</p>

        <div class="dodont-grid">
          <div class="dodont-card dodont-dont">
            <div class="dodont-label">✕ Reduce la asociación</div>
            <ul class="dodont-list">
              <li>Trabajar o revisar correos.</li>
              <li>Ver televisión o series durante horas.</li>
              <li>Navegar redes sociales antes de intentar dormir.</li>
              <li>Comer en la cama de forma habitual.</li>
              <li>Quedarse dando vueltas más de 20 minutos sin poder dormir.</li>
            </ul>
          </div>
        </div>

        <p><strong>La regla de los 20 minutos.</strong> Si después de aproximadamente 20 minutos en la cama no puedes dormirte, la recomendación clínica clásica es: levantarse, ir a otra habitación, hacer una actividad calmante con luz tenue (lectura, por ejemplo), y volver a la cama solo cuando aparezca somnolencia real. Quedarse en la cama frustrado contradice la asociación que queremos construir.</p>

        <div class="info-box">
          <p><strong>Por qué la regla funciona:</strong> evita que el cerebro asocie la cama con "estar despierto y frustrado". Con el tiempo, esta consistencia entrena al cerebro a registrar la cama como una señal específica de "es hora de dormir" — y la transición al sueño se vuelve más natural.</p>
        </div>

        <div class="warn-box">
          <div class="warn-box-icon">⚠️</div>
          <p><strong>Importante:</strong> si tienes dificultades persistentes para conciliar el sueño que no responden a estos cambios después de varias semanas de aplicación consistente, considera consultar a un profesional de salud. La terapia cognitivo-conductual para el insomnio (TCC-I) es una intervención profesional validada que un especialista puede aplicar de manera personalizada.</p>
        </div>
      </div>
    </div>
  </div>

      <div class="acc-body-inner">
        <p>Este plan de 7 días es un <strong>marco educativo de aplicación progresiva</strong>. No es una receta universal — es una secuencia ordenada para empezar a integrar los conceptos de los módulos anteriores sin intentar implementarlos todos al mismo tiempo.</p>
        <p>Lo que cambia entre semanas es la cantidad de elementos integrados. Lo que se mantiene a lo largo de todos los días es el principio central: <strong>la hora de despertar es el ancla más estable del ciclo</strong>. Es el primer cambio que se introduce, y se sostiene durante los siete días.</p>

        <div class="info-box">
          <p><strong>Antes de empezar:</strong> los efectos varían entre personas. Algunas notan diferencia en pocos días, otras necesitan más tiempo. Los primeros días pueden resultar incómodos mientras el cuerpo se adapta a horarios distintos. La consistencia es lo que importa — no la perfección de un solo día.</p>

<script>
  var openItems = {};
  var completed = {};
  var totalSections = 7;

  function toggle(id) {
    var item = document.getElementById(id);
    var body = document.getElementById('body-' + id);
    var isOpen = openItems[id];
    if (isOpen) {
      body.style.maxHeight = '0px';
      item.classList.remove('open');
      openItems[id] = false;
    } else {
      body.style.maxHeight = body.scrollHeight + 'px';
      item.classList.add('open');
      openItems[id] = true;
      if (!completed[id]) {
        setTimeout(function() { markRead(id); }, 1800);
      }
    }
    updateXP();
  }

  function markRead(id) {
    if (completed[id]) return;
    completed[id] = true;
    document.getElementById(id).classList.add('completed');
    updateXP();
  }

  function updateXP() {
    var count = Object.keys(completed).length;
    var pct = Math.round((count / totalSections) * 100);
    document.getElementById('xpFill').style.width = pct + '%';
    document.getElementById('xpPct').textContent = pct + '%';
  }

  function completarCap() {
    for (var i = 1; i <= totalSections; i++) {
      completed['s' + i] = true;
      document.getElementById('s' + i).classList.add('completed');
    }
    document.getElementById('xpFill').style.width = '100%';
    document.getElementById('xpPct').textContent = '100%';
    var toast = document.getElementById('toast');
    toast.classList.add('show');
    setTimeout(function() { toast.classList.remove('show'); }, 3000);
  }

  function toggleCheck(el) {
    el.classList.toggle('checked');
  }

  setTimeout(function() {
    document.getElementById('xpFill').style.width = '0%';
  }, 100);
</script>
