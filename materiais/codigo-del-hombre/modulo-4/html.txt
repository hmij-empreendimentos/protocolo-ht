  <!-- S1 - LAS ULTIMAS 2 HORAS -->
  <div class="acc-item" id="s1">
    <button class="acc-trigger" onclick="toggle('s1')">

      <div class="acc-titles">
        <span class="acc-section-label">La ventana crítica</span>
        <div class="acc-titulo">Las últimas 2 horas del día</div>
      </div>
      <div class="acc-right">
        <div class="acc-arrow-btn"><svg viewBox="0 0 12 12"><polyline points="2,4 6,8 10,4"/></svg></div>
        <span class="acc-check">✅</span>
      </div>
    </button>
    <div class="acc-body" id="body-s1">
      <div class="acc-body-inner">
        <p>Las <strong>2 horas anteriores al sueño</strong> son la ventana donde más se decide la calidad del descanso nocturno. Lo que ocurre durante ese período tiene impacto desproporcionado: actividades estimulantes mantienen al cuerpo en modo de alerta, mientras que actividades calmas le indican al sistema nervioso que el día está cerrando.</p>
        <p>Matthew Walker, profesor de UC Berkeley, ha descrito en <em>Why We Sleep</em> (2017) cómo el cuerpo necesita una <strong>transición gradual</strong> entre estados. No es posible pasar de un día intenso a un sueño profundo en cinco minutos. La transición se construye en los detalles de la última hora y media.</p>

        <div class="quote">
          <div class="quote-text">"El sueño no es algo que enciendes — es algo que invitas. Y la invitación empieza horas antes de acostarte."</div>
          <cite class="quote-cite">— Investigación sobre rutinas pre-sueño</cite>
        </div>

        <p>Lo que la investigación documenta como apoyo a una buena transición al sueño:</p>

        <div class="dodont-grid">
          <div class="dodont-card dodont-do">
            <div class="dodont-label">✓ Apoya la transición</div>
            <ul class="dodont-list">
              <li>Bajar la intensidad de las luces — usar iluminación tenue y cálida.</li>
              <li>Reducir actividades cognitivamente exigentes (trabajo, decisiones importantes).</li>
              <li>Bajar la temperatura del ambiente — el cuerpo se prepara mejor para dormir en un entorno fresco.</li>
              <li>Actividades calmas: lectura física, conversación tranquila, estiramientos suaves.</li>
              <li>Preparar el día siguiente — reduce la actividad mental de "cosas por resolver".</li>
            </ul>
          </div>
          <div class="dodont-card dodont-dont">
            <div class="dodont-label">✕ Dificulta la transición</div>
            <ul class="dodont-list">
              <li>Revisar correo o mensajes de trabajo en las últimas horas.</li>
              <li>Ver noticias o contenido emocionalmente activador.</li>
              <li>Sostener conversaciones difíciles o discusiones intensas.</li>
              <li>Ejercicio muy intenso (ya cubierto en el Módulo 02).</li>
              <li>Cenas pesadas (ya cubierto en el Módulo 03).</li>
            </ul>
          </div>
        </div>

        <div class="info-box">
          <p><strong>La idea central:</strong> las últimas 2 horas funcionan como una "puerta de entrada" al sueño. No se trata de hacer todo perfecto cada noche — se trata de no <em>contradecir</em> activamente la transición natural que el cuerpo intenta hacer.</p>
        </div>
      </div>
    </div>

  <!-- S2 - LUZ AZUL -->
  <div class="acc-item" id="s2">
    <button class="acc-trigger" onclick="toggle('s2')">
      <div class="acc-icon-wrap">📱</div>
      <div class="acc-titles">
        <span class="acc-section-label">Luz de pantallas</span>
        <div class="acc-titulo">Por qué la luz azul es problemática</div>
      </div>
      <div class="acc-right">
        <div class="acc-arrow-btn"><svg viewBox="0 0 12 12"><polyline points="2,4 6,8 10,4"/></svg></div>
        <span class="acc-check">✅</span>
      </div>
    </button>
    <div class="acc-body" id="body-s2">
      <div class="acc-body-inner">
        <p>La luz emitida por celulares, computadoras, tablets y televisores contiene una proporción significativa de <strong>luz de longitud de onda corta</strong> — la llamada "luz azul". Investigación desarrollada en el Brigham and Women's Hospital, afiliado a la Harvard Medical School, documenta que la exposición a esta luz en horas nocturnas <strong>reduce significativamente la señal natural de descanso</strong> que el cuerpo prepara en ese horario.</p>
        <p>Charles Czeisler y su equipo, en estudios publicados en <em>The Journal of Clinical Endocrinology & Metabolism</em> (2011), midieron el efecto: <strong>la exposición a luz brillante en la noche puede retrasar el reloj biológico interno de manera mensurable</strong>. El cerebro interpreta esa luz como una extensión del día, postergando la señal de descanso.</p>

        <div class="quote">
          <div class="quote-text">"La exposición a la luz de pantallas en las horas previas a dormir es uno de los disruptores más comunes y subestimados del sueño moderno."</div>
          <cite class="quote-cite">— Investigación en Harvard Medical School</cite>
        </div>

        <p>Por qué la luz de pantallas afecta el sueño:</p>

        <div class="solucion-card">
          <div class="solucion-num">01</div>
          <div class="solucion-info">
            <div class="solucion-titulo">El cerebro interpreta luz brillante como "día"</div>
            <div class="solucion-desc">El sistema circadiano usa la luz como referencia principal para regular el ciclo de alerta y descanso. Luz brillante en la noche envía la señal contraria a la que el cuerpo necesita.</div>
          </div>
        </div>

        <div class="solucion-card">
          <div class="solucion-num">02</div>
          <div class="solucion-info">
            <div class="solucion-titulo">Retrasa el inicio natural del sueño</div>
            <div class="solucion-desc">Cuando la señal de descanso se retrasa, el sueño tarda más en llegar — aunque la persona se sienta cansada.</div>
          </div>
        </div>

        <div class="solucion-card">
          <div class="solucion-num">03</div>
          <div class="solucion-info">
            <div class="solucion-titulo">Mantiene el sistema en modo de alerta</div>
            <div class="solucion-desc">No solo la luz: el <em>contenido</em> de pantallas (redes sociales, noticias, conversaciones) mantiene el sistema cognitivo activo, dificultando la desaceleración necesaria.</div>
          </div>
        </div>

        <div class="solucion-card">
          <div class="solucion-num">04</div>
          <div class="solucion-info">
            <div class="solucion-titulo">Reduce el sueño profundo posterior</div>
            <div class="solucion-desc">Estudios documentan que la exposición a luz de pantallas antes de dormir se asocia con menor proporción de sueño profundo (fase N3) en la noche siguiente.</div>
          </div>
        </div>

        <p>Estrategias educativas para reducir el impacto:</p>

        <div class="solucion-card">
          <div class="solucion-num">A</div>
          <div class="solucion-info">
            <div class="solucion-titulo">Lo ideal — sin pantallas 1 a 2 horas antes</div>
            <div class="solucion-desc">La estrategia más efectiva, según la literatura. Un libro físico, una conversación, un podcast con los ojos cerrados son alternativas que no involucran exposición lumínica de pantalla.</div>
          </div>
        </div>

        <div class="solucion-card">
          <div class="solucion-num">B</div>
          <div class="solucion-info">
            <div class="solucion-titulo">Si necesitas usar pantallas — filtros de pantalla</div>
            <div class="solucion-desc">Activa "Night Shift" (iOS), "Filtro de luz azul" (Android) o configuraciones equivalentes en computadora. Reducen — aunque no eliminan — la fracción de luz azul.</div>
          </div>
        </div>

        <div class="solucion-card">
          <div class="solucion-num">C</div>
          <div class="solucion-info">
            <div class="solucion-titulo">Gafas con filtro ámbar</div>
            <div class="solucion-desc">Las gafas con lentes amarillas o ámbar bloquean parte significativa de la luz azul. Pueden ser útiles para quienes deben mantener uso de pantallas en horas nocturnas por trabajo.</div>
          </div>
        </div>

        <div class="solucion-card">
          <div class="solucion-num">D</div>
          <div class="solucion-info">
            <div class="solucion-titulo">Iluminación cálida del hogar</div>
            <div class="solucion-desc">Bombillas con temperatura de color cálida (alrededor de 2700K o menos) en lugar de luz blanca/fría. Crea un ambiente nocturno que apoya la transición — no la contradice.</div>
          </div>
        </div>

        <div class="warn-box">
          <div class="warn-box-icon">⚠️</div>
          <p><strong>Importante:</strong> el modo nocturno de pantallas reduce parcialmente la luz azul, pero <em>no la elimina</em>. Además, el contenido de pantallas (redes sociales, mensajes de trabajo, noticias) sigue activando el sistema cognitivo. La estrategia más efectiva sigue siendo reducir el uso de pantallas en las horas previas al sueño.</p>
        </div>
      </div>
    </div>
  </div>

  <!-- S3 - TECNICAS DE RELAJACION -->
  <div class="acc-item" id="s3">
    <button class="acc-trigger" onclick="toggle('s3')">
      <div class="acc-icon-wrap">🫁</div>
      <div class="acc-titles">
        <span class="acc-section-label">Técnicas educativas</span>
        <div class="acc-titulo">4 prácticas de relajación</div>
      </div>
      <div class="acc-right">
        <div class="acc-arrow-btn"><svg viewBox="0 0 12 12"><polyline points="2,4 6,8 10,4"/></svg></div>
        <span class="acc-check">✅</span>
      </div>
    </button>
    <div class="acc-body" id="body-s3">
      <div class="acc-body-inner">
        <p>El sistema nervioso humano tiene dos ramas principales: una asociada con <em>activación</em> (la respuesta de alerta y acción) y otra asociada con <em>descanso</em> (el sistema parasimpático). Para dormir bien, el cuerpo necesita transicionar al sistema de descanso. Las técnicas a continuación son herramientas educativas que la investigación sobre relajación documenta como útiles para apoyar esa transición.</p>

        <p>Cuatro técnicas con buena base académica:</p>

        <!-- TECNICA 1 - 4-7-8 -->
        <div class="tecnica-card t1">
          <div class="tecnica-header">
            <div class="tecnica-emoji">🫁</div>
            <div class="tecnica-info">
              <div class="tecnica-tag">Técnica 01 · Respiración</div>
              <div class="tecnica-titulo">Respiración 4-7-8</div>
            </div>
          </div>
          <div class="tecnica-intro">Técnica difundida por el Dr. Andrew Weil, médico y profesor de la University of Arizona. Es una práctica de respiración diafragmática que utiliza patrones de inhalación, retención y exhalación para apoyar la transición al estado de descanso.</div>
          <ol class="tecnica-pasos">
            <li>Siéntate o acuéstate cómodamente. Exhala completamente por la boca.</li>
            <li>Inhala por la nariz contando hasta <strong>4</strong>.</li>
            <li>Mantén el aire contando hasta <strong>7</strong>.</li>
            <li>Exhala lentamente por la boca contando hasta <strong>8</strong>.</li>
            <li>Repite el ciclo completo entre 3 y 4 veces.</li>
          </ol>
        </div>

        <!-- TECNICA 2 - RELAJACION MUSCULAR PROGRESIVA -->
        <div class="tecnica-card t2">
          <div class="tecnica-header">
            <div class="tecnica-emoji">💪</div>
            <div class="tecnica-info">
              <div class="tecnica-tag">Técnica 02 · Cuerpo</div>
              <div class="tecnica-titulo">Relajación Muscular Progresiva</div>
            </div>
          </div>
          <div class="tecnica-intro">Técnica desarrollada por Edmund Jacobson en la University of Chicago en la década de 1930. Consiste en tensar y relajar grupos musculares específicos para entrenar al cuerpo a reconocer la diferencia entre tensión y relajación.</div>
          <ol class="tecnica-pasos">
            <li>Acuéstate boca arriba en un lugar cómodo, cierra los ojos.</li>
            <li>Empieza por los pies: tensa durante <strong>5 segundos</strong>, luego suelta durante <strong>10 segundos</strong>.</li>
            <li>Sube progresivamente: pantorrillas, muslos, abdomen, pecho, brazos, manos, cuello y rostro.</li>
            <li>Al terminar, permanece quieto durante <strong>2 minutos</strong>, observando la sensación de relajación.</li>
          </ol>
        </div>

        <!-- TECNICA 3 - BRAIN DUMP -->
        <div class="tecnica-card t3">
          <div class="tecnica-header">
            <div class="tecnica-emoji">📝</div>
            <div class="tecnica-info">
              <div class="tecnica-tag">Técnica 03 · Mente</div>
              <div class="tecnica-titulo">Brain Dump (Vaciado Mental)</div>
            </div>
          </div>
          <div class="tecnica-intro">Práctica respaldada por la investigación de James Pennebaker, profesor de psicología en la University of Texas at Austin, sobre el efecto del escribir como herramienta para reducir la rumiación mental — los pensamientos repetitivos que dificultan conciliar el sueño.</div>
          <ol class="tecnica-pasos">
            <li>Toma una hoja de papel y un bolígrafo — no el celular.</li>
            <li>Escribe todo lo que esté en tu cabeza: tareas pendientes, preocupaciones, ideas.</li>
            <li>No organices ni priorices — simplemente vacía.</li>
            <li>Cuando termines, cierra el cuaderno. La información ya no está dando vueltas en tu mente — está registrada.</li>
          </ol>
        </div>

        <!-- TECNICA 4 - BODY SCAN -->
        <div class="tecnica-card t4">
          <div class="tecnica-header">
            <div class="tecnica-emoji">🧘</div>
            <div class="tecnica-info">
              <div class="tecnica-tag">Técnica 04 · Atención</div>
              <div class="tecnica-titulo">Body Scan (Escaneo Corporal)</div>
            </div>
          </div>
          <div class="tecnica-intro">Práctica documentada en programas de reducción de estrés basados en atención plena, como el MBSR desarrollado por Jon Kabat-Zinn en la University of Massachusetts Medical School. Lleva la atención al cuerpo y aleja la mente de pensamientos rumiantes.</div>
          <ol class="tecnica-pasos">
            <li>Acuéstate, cierra los ojos, respira profundo <strong>3 veces</strong>.</li>
            <li>Lleva la atención a los pies. Nota las sensaciones presentes sin juzgarlas.</li>
            <li>Sube lentamente: tobillos, pantorrillas, rodillas, muslos... hasta llegar a la cabeza.</li>
            <li>Dedica entre <strong>5 y 10 minutos</strong> al recorrido completo.</li>
          </ol>
        </div>

        <div class="info-box">
          <p><strong>Sugerencia educativa:</strong> no es necesario practicar las cuatro técnicas. Elige una o dos que te resulten naturales y úsalas con consistencia. La respiración 4-7-8 es frecuentemente recomendada como primer punto de partida por su simplicidad.</p>
        </div>

        <div class="warn-box">
          <div class="warn-box-icon">⚠️</div>
          <p><strong>Importante:</strong> si durante cualquier técnica de respiración experimentas mareo, ansiedad o incomodidad, suspende la práctica de inmediato y respira con normalidad. Personas con condiciones cardíacas, respiratorias, embarazo o cualquier otra condición particular deben consultar a un profesional de salud antes de adoptar prácticas regulares de respiración o relajación.</p>
        </div>
      </div>
    </div>
  </div>

      <div class="acc-right">

        <span class="acc-check">✅</span>

    <div class="acc-body" id="body-s4">
      <div class="acc-body-inner">
        <p>El protocolo a continuación es un <strong>marco educativo</strong> que integra todo lo visto en los módulos anteriores. No es una prescripción — es una guía que puedes adaptar a tu horario, contexto y preferencias.</p>

        <div class="rutina-wrap">
          <div class="rutina-titulo">🌙 Protocolo Nocturno · 6 pasos</div>

          <div class="rutina-step">
            <div class="rutina-step-num">1</div>
            <div class="rutina-step-titulo">Cierre del día</div>
            <div class="rutina-step-desc">Revisa tu agenda del día siguiente. Prepara la ropa. Si tienes pensamientos dando vueltas, haz un brain dump rápido. Esto cierra mentalmente el día.</div>
            <span class="rutina-step-tiempo">⏱ 2h antes · 10 min</span>
          </div>

          <div class="rutina-step">
            <div class="rutina-step-num">2</div>
            <div class="rutina-step-titulo">Bajar luces y desconexión digital</div>
            <div class="rutina-step-desc">Enciende iluminación cálida y tenue. Deja el celular en modo no molestar o en otra habitación. A partir de aquí, idealmente, sin pantallas.</div>
            <span class="rutina-step-tiempo">⏱ 1.5h antes</span>
          </div>

          <div class="rutina-step">
            <div class="rutina-step-num">3</div>
            <div class="rutina-step-titulo">Actividad calmante</div>
            <div class="rutina-step-desc">Lectura física, conversación tranquila, música suave, estiramientos. Cualquier actividad que el sistema nervioso reconozca como "no urgente".</div>
            <span class="rutina-step-tiempo">⏱ 1.5 a 1h antes · 30–45 min</span>
          </div>

          <div class="rutina-step">
            <div class="rutina-step-num">4</div>
            <div class="rutina-step-titulo">Higiene y preparación</div>
            <div class="rutina-step-desc">Ducha tibia (no muy caliente), cepillado de dientes y cuidados personales habituales. Una ducha tibia produce una caída posterior de temperatura corporal que apoya la entrada al sueño.</div>
            <span class="rutina-step-tiempo">⏱ 45–30 min antes · 15 min</span>
          </div>

          <div class="rutina-step">
            <div class="rutina-step-num">5</div>
            <div class="rutina-step-titulo">Snack ligero (opcional)</div>

          </div>
        </div>

          <p><strong>El principio detrás del protocolo:</strong> no necesitas ejecutar los 6 pasos perfectamente cada noche. Lo que importa es la <em>consistencia general</em>.         </div>

<script>
  var openItems = {};
  var completed = {};
  var totalSections = 6;

  function toggle(id) {
    var item = document.getElementById(id);
    var body = document.getElementById('body-' + id);
    var isOpen = openItems[id];
    if (isOpen) {
      body.style.maxHeight = '0px';
      item.classList.remove('open');
      openItems[id] = false;
    } else {
      body.style.maxHeight = body.scrollHeight + 'px';
      item.classList.add('open');
      openItems[id] = true;
      if (!completed[id]) {
        setTimeout(function() { markRead(id); }, 1800);
      }
    }
    updateXP();
  }

  function markRead(id) {
    if (completed[id]) return;
    completed[id] = true;
    document.getElementById(id).classList.add('completed');
    updateXP();
  }

  function updateXP() {
    var count = Object.keys(completed).length;
    var pct = Math.round((count / totalSections) * 100);
    document.getElementById('xpFill').style.width = pct + '%';
    document.getElementById('xpPct').textContent = pct + '%';
  }

  function completarCap() {
    for (var i = 1; i <= totalSections; i++) {
      completed['s' + i] = true;
      document.getElementById('s' + i).classList.add('completed');
    }
    document.getElementById('xpFill').style.width = '100%';
    document.getElementById('xpPct').textContent = '100%';
    var toast = document.getElementById('toast');
    toast.classList.add('show');
    setTimeout(function() { toast.classList.remove('show'); }, 3000);
  }

  function toggleCheck(el) {
    el.classList.toggle('checked');
  }

  setTimeout(function() {
    document.getElementById('xpFill').style.width = '0%';
  }, 100);
</script>
